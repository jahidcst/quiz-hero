  3. What is EventLoop and how it works?

    4. Ans : Event loop is a single one simple job that monitor CALLSTACK and CALLBACKQueue.If the Call Stack is empty, the Event Loop will take the first event from the queue and will push it to the Call Stack, which effectively runs it. Such an iteration is called a tick in the Event Loop.
