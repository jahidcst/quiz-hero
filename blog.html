1. What is the difference between map , filter, forEach and find?

    2. Ans : These four methods are array methods but there have some different things :

    1. Map automatically return a array which actually pass the condition.
    2. Filter return those array which is passed the condition
    3. If the condition is passed in find then it return only the first matching result

  2. What is the difference in localStorage , sessionStorage and cookies?

    3. Ans : localStorage set the value to the localDevice and can use the value in offline. But sessionStorage set the data for temporary and this can't be use in later. Cookies are client-side files that are stored on a local computer and contain user information
